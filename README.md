# Telegram Mini App - Напоминания

Приложение-напоминалка (To-Do список) для Telegram Mini App, созданное с использованием Next.js, React и Tailwind CSS.

## Функции

- ✅ **Главная страница** со списком всех напоминаний
- ✅ **Форма добавления** нового напоминания с опциональной датой/временем
- ✅ **Управление напоминаниями**: отметка выполненного, удаление
- ✅ **Нижнее меню навигации** с тремя вкладками:
  - Все напоминания
  - Выполненные напоминания
  - Профиль пользователя
- ✅ **Сохранение в localStorage** - напоминания сохраняются после обновления страницы
- ✅ **Интеграция с Telegram WebApp SDK** - отображение данных пользователя в профиле
- ✅ **Мобильный дизайн** с использованием Tailwind CSS
- ✅ **Все на русском языке**

## Технологии

- **Next.js 16** - React фреймворк
- **React 19** - UI библиотека
- **TypeScript** - типизация
- **Tailwind CSS 4** - стилизация
- **Telegram WebApp SDK** - интеграция с Telegram

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Запустите dev сервер:
```bash
npm run dev
```

3. Откройте [http://localhost:3000](http://localhost:3000) в браузере

## Структура проекта

```
app/
├── components/          # React компоненты
│   ├── AddReminderForm.tsx    # Форма добавления напоминания
│   ├── BottomNavigation.tsx   # Нижнее меню навигации
│   ├── Profile.tsx            # Страница профиля
│   ├── ReminderItem.tsx       # Элемент списка напоминаний
│   └── RemindersList.tsx      # Список напоминаний

├── types/               # TypeScript типы

├── types.ts             # Типы приложения

```

## Использование в Telegram

Для использования приложения в Telegram Mini App:

1. Создайте бота через [@BotFather](https://t.me/botfather)
2. Используйте команду `/newapp` для создания Mini App
3. Укажите URL вашего развернутого приложения
4. Откройте бота в Telegram и нажмите на кнопку Mini App

## Развертывание

Приложение можно развернуть на:
- **Vercel** (рекомендуется для Next.js)
- **Netlify**
- **Любой другой хостинг**, поддерживающий Next.js

## Особенности

- **Автосохранение**: Все напоминания автоматически сохраняются в localStorage
- **Адаптивный дизайн**: Оптимизирован для мобильных устройств
- **Интеграция с Telegram**: Отображение данных пользователя из Telegram
- **Форматирование дат**: Умное форматирование дат (Сегодня, Завтра, или полная дата)
